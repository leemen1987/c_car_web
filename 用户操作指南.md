# 包车排班系统 - 用户操作指南

## 一、系统登录

1. 打开浏览器，输入系统地址
2. 输入用户名和密码，点击「登录」
3. 默认管理员账号：`admin` / `admin123`

---

## 二、任务管理

任务管理是系统的核心页面，用于录入、排班、完成包车任务以及发起云之家审批。

### 2.1 录入任务

1. 点击页面右上角「**录入任务**」按钮
2. 选择用车类型：
   - **个人**：填写姓名和手机号码
   - **单位**：从下拉列表选择用车单位和联系人
3. 填写以下信息：
   - 车辆类型（大巴/中巴/小巴/商务车）
   - 出发地点、目的地
   - 出车时间、回程时间（系统自动计算租用天数）
   - 任务里程、租车费
   - 油电费、桥路费（预估）
   - 差费标准（自动计算司机人工费）
4. 系统自动计算：预计成本 = 油电费 + 桥路费 + 人工费
5. 系统自动计算：预估利润 = 租车费 - 预计成本
6. 点击「确定」保存

### 2.2 排班分配

任务录入后状态为「**待排班**」，需要分配车辆和司机：

1. 在任务列表中找到待排班的任务，点击「**排班分配**」
2. 弹窗显示该时间段可用的车辆和司机
3. 选择车辆和司机后，点击「**确认排班**」
4. 任务状态变为「**已排班**」

> 系统会自动排除该时间段已被占用的车辆和司机。

### 2.3 完成任务

任务执行完毕后，录入实际费用：

1. 在已排班任务中点击「**完成任务**」
2. 填写实际费用：
   - 实际油电费、实际桥路费、实际司机人工费、其他费用
3. 系统自动计算：实际成本、最终利润
4. 点击「**确认完成**」
5. 任务状态变为「**已完成**」

### 2.4 编辑和删除

- 未完成的任务可以「**编辑**」修改信息
- 修改记录会自动保存到「**变更记录**」中
- 可以「**删除**」不需要的任务

### 2.5 任务状态说明

| 状态 | 颜色 | 说明 |
|------|------|------|
| 待排班 | 橙色 | 已录入但未分配车辆司机 |
| 已排班 | 蓝色 | 已分配车辆司机，可发起审批 |
| 已完成 | 绿色 | 已执行完毕并录入实际费用 |

## 三、发起云之家审批

### 3.1 操作步骤

1. 点击任务管理页面的「**发起审批**」按钮（绿色）
2. 弹窗中显示所有**已排班**的任务列表
3. 勾选需要发起审批的任务（只能选同一所属公司的车辆）
4. 底部显示已选任务数量和所属公司分布
5. 点击「**发起审批**」提交

### 3.2 审批规则

| 规则 | 说明 |
|------|------|
| 只能选已排班任务 | 待排班和已完成的任务无法勾选 |
| 已过期任务不能发起 | 出车时间已过的任务自动置灰 |
| 已发起的任务不能重复 | 已提交审批的任务显示「已发起」，无法再勾选 |
| 不能混合不同公司 | 国顺司、国开司、外单位的车辆必须分开发起 |

### 3.3 审批模板自动匹配

系统根据车辆的**所属公司**自动选择对应的审批模板：

| 车辆所属公司 | 审批模板 |
|------------|---------|
| 国顺司 | 申请使用国顺司车辆测试 |
| 国开司 | 申请使用国开司车辆测试 |
| 外单位 | 申请使用外单位司车辆测试 |

### 3.4 审批状态显示

任务管理页面的「审批」列显示：

| 状态 | 说明 |
|------|------|
| - | 未发起审批 |
| 已发起（绿色） | 审批已提交，等待审批人处理 |
| 已通过（蓝色） | 审批已通过 |
| 已拒绝（红色） | 审批被拒绝 |

下方还会显示审批流水号（如 `SQSYGSSCLCS-20260527-005`），可在云之家APP中通过该流水号搜索审批单。

## 四、车辆管理

管理公司所有车辆信息。

### 4.1 添加车辆

1. 点击「**添加车辆**」
2. 填写车牌号、选择车辆类型
3. **选择所属公司**（国顺司 / 国开司 / 外单位）
4. 选择状态（可用 / 忙碌 / 维修中）
5. 点击「确定」

> **重要：** 所属公司决定了该车辆发起审批时使用哪个模板，请务必正确选择。

### 4.2 编辑车辆

点击「编辑」可修改车辆信息，包括所属公司。

---

## 五、司机管理

管理司机信息（姓名、电话、状态）。

### 5.1 添加司机

1. 点击「添加司机」
2. 填写姓名和电话号码
3. 选择状态（可用 / 忙碌 / 离职）

---

## 六、客户管理

管理用车单位和联系人信息。

### 6.1 添加单位

1. 点击「添加单位」

### 6.2 管理联系人

在单位列表中点击「联系人」，可为每个单位添加多个联系人。

## 七、差费标准

2. 填写目的地名称
3. 填写费用金额和对应天数
4. 系统在排班时自动计算人工费

## 九、常见问题

### Q: 任务录入后看不到？

A: 检查任务状态筛选条件，确保显示了「待排班」状态的任务。

### Q: 排班时提示没有可用车辆？

A: 检查车辆状态是否为「可用」，以及该时间段是否与其他任务冲突。

### Q: 发起审批报错？

A: 确认以下几点：
- 车辆已设置所属公司
- 任务状态为「已排班」
- 出车时间未过期
- 所选任务为同一所属公司
- 云之家服务正常

### Q: 审批流水号在哪里查看？

A: 在任务管理页面的「审批」列中，已发起审批的任务会显示流水号。也可以在云之家APP中通过流水号搜索。

### Q: 审批被拒绝了怎么办？

A: 审批拒绝后，任务的审批状态会变为「已拒绝」，可以重新发起审批。

### Q: 页面长时间不操作后报错？

A: 这是正常的连接超时，刷新页面即可恢复。

录入任务 → 待排班
    ↓
排班分配 → 已排班

    ↓
审批通过 → 任务可执行
    ↓
